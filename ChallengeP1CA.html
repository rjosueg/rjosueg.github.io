Common CSS Properties of the Box Model

Width, Height, Padding and more!

Property Name Description
height:; Height The height property sets the height of an element.It does not include padding, borders, or margins!
width:; Width The width property sets the width of an element. It does not include padding, borders, or margins!
padding:; Padding It is used to generate space around an element's content, inside of any defined borders.
border:; Border This property allows you to specify the style, width, and color of an element's border.
margin:; Margin Used to create space around elements, outside of any defined borders.
